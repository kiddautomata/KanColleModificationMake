一、使用前说明

  • 1、本模版为 DMM网页游戏 舰队收藏 提供缓存替换用图片的编辑模版，请勿用作其他用途

  • 3、本模版推荐使用 谷歌浏览器

三、制作流程

  • ♦ 点击 魔改文件制作页面 进入魔改文件制作，此页面一共可以制作除语音文件外所有上面提及的文件
  • ♦ 制作流程分为：选择舰船 → 选择改造等级 → 上传立绘 → 根据场景进行制作 → 导出zip压缩包
  • ♦ 可以导出两种类型的压缩包 cache.zip 和 bat.zip 具体内容介绍见 七、下载部分
  • ♦ 点击 图片文件批处理 可以将bat.zip中解压出的文件夹 进行上传 打包成缓存格式的文件夹
  • ♦ 点击 语音文件批处理 可以将自定义语音文件的文件夹 进行上传 打包成缓存格式的文件夹

四、上传部分

  • ♦ 点击制作后用户首先需要选择舰船，可以使用下拉框选择，也可以使用搜索框搜索
  • ♦ 选择舰船后会出现舰船改造等级的选择框，必须选中一项，其余可以根据制作需求勾选
  • ♦ 用户至少需要上传一张全身立绘来进行后续流程，其余上传部分根据需求可选
  • ♦ 可上传部分为：
  •   全身正常立绘（必选）
      全身中破立绘（可选）
      两倍全身正常（可选）
      两倍全身中破（可选）
      特殊攻击立绘（可选）
  • ♦ 如果需修改立绘，进入制作界面后需要点击最上部分的“全身”场景按钮重新上传立绘

五、操作部分

  • ★ 每个制作场景图片完成后必须点击保存（重要！）！未保存将视为未制作！
  • ♦ 最上部分单选框为：素材制作场景切换
  • ♦ 最下部分单选框为：角色改造等级切换
  • ♦ 最下部分点击查看可以确认当前制作进度
  • ♦ 根据页面左边和右边进行设置，鼠标左键可以按住人物角色进行拖动
  • ♦ 鼠标滚轮上下控制大小，最大100%，最小0%，改变间隔为1像素
  • ♦ 鼠标滚轮左右控制旋转角度，最大180°，最小-180°，改变间隔为1像素
  • ♦ 数值框可以输入小数精确改变数值（坐标文件输入框必须使用整数）
  • ♦ 选中的数值框可以使用鼠标滚轮改变数值，改变间隔为1像素

六、文字部分

  • ♦ 舰名必须为日文汉字
  • ♦ 假名必须为日文平甲或片甲
  • ♦ 汉字特殊符号 ： / 换行 ○ 特殊形状 - 特殊符号

七、下载部分

  • ♦ 下载为cache.zip的文件可直接作为缓存文件使用
  • ♦ cache.zip文件，解压后可直接替换游戏缓存
  • ♦ 下载为bat.zip的文件可作为批处理文件使用
  • ♦ bat.zip文件，将所有文件放入一个文件夹内方便管理
  • ♦ bat文件夹格式为:ship/舰船ID/png文件
  • 例：
  •  ship（舰船图片 主文件夹）
      0000（舰船角色ID 文件夹）
       full_.png（舰船全身正常 立绘文件）
       full_dmg.png（舰船全身中破 立绘文件）
       config.ini（舰船立绘坐标文件）

  • ♦ 语音文件夹格式为 sound/舰船ID/mp3文件
  • 例：
  •  sound（舰船图片 主文件夹）
      0000（舰船角色ID 文件夹）
       1.mp3（获得/登陆游戏 语音文件）
       2.mp3（母港/详细阅览 语音文件）
       3.mp3（母港/详细阅览2 语音文件）

  • ♦ 如需要作为缓存文件使用，上传至批处理页面导出即可
